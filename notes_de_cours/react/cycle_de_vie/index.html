    Le cycle de vie d'un composant

    Chaque composante a un cycle de vie qu’on peut diviser en trois grandes étapes :

    • Chargement (Mounting) : C’est ici que les composantes sont ajoutées au DOM et affichées dans la page.
    • Mise-à-jour (Updating) : Cette étape est lancée quand l’objet state ou props de la composante est modifié.
    • Déchargement (Unmounting) : Si on enlève une composante du DOM, cette étape est alors effectuée.

    Chargement (Mounting)

    Il y a quatres méthodes qui sont exécutés dans cette ordre lors du chargement

    • constructor(props) : Le constructeur de la classe, on y défini entre autre les valeurs initiales de l’objet state.
    • getDerivedStateFromProps(props, state) : Appelé juste avant d’afficher la composante dans le DOM.
    • render() : Affiche la composante dans la page.
    • componentDidMount() : Appelé après avoir affiché la composante dans le DOM. On y place le traitement à faire une fois que la composante fait partie du DOM.
